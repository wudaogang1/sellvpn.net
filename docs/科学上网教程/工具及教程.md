# 一、什么是科学上网

**科学上网**是指通过技术手段访问全球互联网资源的一种方式。

通常用于访问：

- Google  
- YouTube  
- GitHub  
- Reddit  
- 各类海外网站  

核心原理：

你的设备 → 代理服务器 → 目标网站

通过代理服务器转发网络请求，从而绕过网络限制。

---

## 常见科学上网方式

目前主流技术包括：

- **VPN（虚拟专用网络）**
- **代理服务器（HTTP / SOCKS5）**
- **Shadowsocks**
- **V2Ray**
- **Trojan**

这些技术本质上都是通过 **远程服务器中转网络流量** 来实现访问外网。

---

# 二、什么是翻墙

**翻墙**是“科学上网”的通俗说法。

“墙”指的是网络限制系统。

因此：

- **翻墙 = 科学上网**
- 只是叫法不同，本质相同。

---

# 三、什么是梯子

**梯子**是科学上网工具的俗称。

通常指：

- VPN 软件  
- Clash  
- Shadowrocket  
- V2Ray 客户端  

例如：

> “有没有好用的梯子？”

其实就是在寻找 **科学上网工具**。

---

# 四、什么是机场

在科学上网领域，**机场**是提供代理节点的服务商。

机场通常提供：

- Shadowsocks 节点  
- V2Ray 节点  
- Trojan 节点  

使用步骤通常为：

1 注册账号  
2 购买套餐  
3 获取订阅 URL  
4 导入客户端使用

# 五、什么是软路由

**软路由**是指通过电脑或小主机运行的路由系统。

常见系统：

- OpenWrt  
- iKuai  
- pfSense  

软路由常用于：

- 科学上网分流  
- 广告过滤  
- 家庭网络管理  

在科学上网中，通常会安装：

- OpenClash  
- PassWall  

这样可以实现 **全家设备统一科学上网**。

# 七、机场和自建节点区别

很多用户会问：

**是用机场还是自己搭服务器？**

## 机场

优点：

- 简单  
- 节点多  
- 适合新手  

缺点：

- 需要付费  

适合：

**大多数普通用户**

---

## 自建节点

优点：

- 完全可控  
- 隐私更好  

缺点：

- 技术门槛高  
- 维护复杂  

适合：

**技术用户**
