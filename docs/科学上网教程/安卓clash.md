## 配置

![Android Clash配置界面，点击配置按钮](https://image.ermao.net/images/article/eh8f4n86/image.png)

点击配置

![Android Clash添加配置，点击右上角加号](https://image.ermao.net/images/article/eh8f4n86/image-1.png)

点击右上角的“+”号

![Android Clash选择URL导入方式](https://image.ermao.net/images/article/eh8f4n86/image-2.png)

点击“URL”

![Android Clash粘贴订阅URL链接](https://image.ermao.net/images/article/eh8f4n86/image-3.png)

将获取到的 URL 粘贴到这里，点击确定

如果你没有 URL，可以在这里获取：[便宜好用的翻墙机场推荐评测](/posts/vpn)

![Android Clash设置自动更新时间](https://image.ermao.net/images/article/eh8f4n86/image-4.png)

自动更新随便设置，设置完成点击确定

![Android Clash保存配置按钮](https://image.ermao.net/images/article/eh8f4n86/image-5.png)

完成后点击右上角的保存按钮

![Android Clash选择保存的配置](https://image.ermao.net/images/article/eh8f4n86/image-6.png)

保存完成后选中刚才保存的配置
